# Workflows

1. **Project Initialization**: Setup Flutter boilerplate, integrate Firebase (google-services.json), and configure dependencies in pubspec.yaml.
2. **Data Modeling**: Create Dart class representations for Firestore collections (UserModel, ServiceModel, BookingModel).
3. **Authentication**: Build UI and Firebase Auth logic for Customer and Admin registration and login.
4. **User Module Implementation**: Build service catalog page, booking form (inject server timestamp), and real-time queue live tracking page via StreamBuilder.
5. **Admin Module Implementation**: Build dashboard for service management (CRUD) and queue management (FCFS Query for confirmation and status updates).
